# Math Tutor - Android 15+

A kid-friendly math game for ages around 7 that introduces:

- Addition
- Subtraction
- Multiplication
- Division

## Concept

The app is designed as a puzzle game:

- Kids choose an operation mode.
- A colorful puzzle card shows a math challenge.
- Four answer buttons make it feel like a mini-quiz game.
- Correct answers increase score and streak.
- Instant feedback keeps the experience playful and encouraging.

## Tech stack

- Kotlin
- Jetpack Compose
- Android SDK 35 (Android 15)

## Build notes

- `minSdk = 35`
- `targetSdk = 35`
- `compileSdk = 35`

This means the app runs on Android 15 and above.
